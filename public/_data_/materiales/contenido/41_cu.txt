CRUCIGRAMA
------------------------------------
arreglos y matrices

1. Atributo de un arreglo que se usa para obtener su tamaño -> length
2. Estructura de control usada generalmente para recorrer arreglos y matrices -> FOR
3. (En inglés) Famoso algoritmo usado para ordenar un arreglo cuya complejidad es n^2 -> bubblesort
4. (En inglés) Famoso algoritmo usado para ordenar un arreglo cuya complejidad es nlog(n)  -> quicksort
5. Caracteristica que se debe especificar para declarar un arreglo o matriz -> tipodedato
6. Propiedad que se debe definir en el momento de crear un arreglo en JAVA -> tamaño
7. Posicion usada para acceder a los elementos de un arreglo o matriz -> indice
8. Triangular de una matriz cuyos elementos estan en y encima de la diagonal principal-> superior
9. Tipo de matriz cuyo tamaño de filas es el mismo que el de sus columnas -> cuadrada
10. Algoritmo de busqueda muy conocido para encontrar la posición de un elemento en un arreglo, el cual requiere que el algoritmo esté previamente ordenado->binaria



SOPA DE LETRAS
----------------------------
funciones y recursividad

1. Tipo de paso de parametros a una funcion en donde si su valor es cambiado en la función este no se alterará en la función principal -> valor
2. Tipo de paso de parametros a una funcion en donde si su valor es cambiado en la función este tambien se alterará en la función principal -> referencia
3. Tecnica de programación en donde una función se llama así misma -> recursividad
4. Sentencia usada en JAVA para devolver un valor desde una función -> return
5. Palabra reservada de JAVA para indicar que una función no retorna ningun valor -> void
6. Variable que es recibida por una función para determinar su comportamiento -> parametro
7. nombre de la función principal en donde inicia un programa en JAVA -> Main
8. Bloque de memoría en donde se almacenan los llamados recursivos de una función -> pila
9. Condición de parada o terminación de una recursion -> caso base
